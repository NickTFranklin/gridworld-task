Instructions: Game 2

In this next game, you will use what you have learned from the first game to guess where you think the goal is.


You won't see the colors we used in the previous game again. All of the colors will be new to you.


On each trial, we will tell you which four keys you can move around the maze. We will either tell you to use the a, s, d and f keys or to use the j, k, l and ; keys. These keys will move you around the maze in the same way that you learned in the first game.
